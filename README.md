# Weather-app
Une application web moderne affichant la météo actuelle et les prévisions sur 5 jours, utilisant l'API OpenWeatherMap. Développée en HTML, CSS et JavaScript vanilla, elle met en avant des compétences en intégration d'APIs REST, gestion d'erreurs, interface responsive, et fonctionnalités avancées comme la géolocalisation et le mode sombre.
